# React Custom Hook Assignment

This project demonstrates the creation and usage of a custom React hook `useFetch` to fetch data from an API.

## Features
- **Custom Hook**: `useFetch` handles data fetching, loading states, and error handling.
- **Premium UI**: A clean, modern interface to display products.
- **Responsive Design**: Adapts to various screen sizes.

## Project Structure
- `src/hooks/useFetch.js`: The custom hook implementation.
- `src/components/ProductList.jsx`: Component that consumes the hook.
- `src/index.css`: Global styles.

## How to Run Locally
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
